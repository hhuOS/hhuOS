ps
=====
Print information about active processes

Usage
-----
```
ps [OPTION]...
```

Supported options:
* -h, --help: Show this help message and exit.

Information about all active processes on the system is printed to standard out in tabular form.

Examples
--------
```
[/]> ps
PID     Threads Name
0       9       Kernel
1       1       /bin/shell
2       1       /bin/ps
```

echo
=====
A simple application, that writes all given arguments to standard output.  
Each argument is treated as a string and printed in the order they are provided,
separated by spaces.

Usage
-----
```sh
echo [STRING]...
```

Supported options:
 * -h/--help: Show this help message and exit.

Examples
--------
```sh
[/]> echo Hello, World!
Hello, World!
[/]> echo This is a simple echo application.
This is a simple echo application.
```

classicube
=====
ClassiCube is a custom Minecraft Classic compatible client written in C from scratch.
See [https://github.com/ClassiCube/ClassiCube](https://github.com/ClassiCube/ClassiCube) for more information.

Usage
-----
```
classicube --singleplayer
```

The hhuOS port of ClassiCube only supports software rendering, and network and sound are disabled.
